Miroute - Aplicação para Controle da Qualidade do Leite

Descrição

Miroute é uma aplicação desenvolvida para rastrear a qualidade do leite, monitorando todos os dados de transporte e otimização dos processos. O sistema coleta e processa dados sobre a qualidade do leite e oferece visualizações e relatórios em tempo real para as partes interessadas. O objetivo principal é garantir a qualidade do leite desde a fazenda até o consumidor final.

Tecnologias Utilizadas

  • Python
  • Django
  • PostgreSQL
  • IoT (Internet das Coisas)
  • APIs RESTful
